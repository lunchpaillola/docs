## What is Daily?

[Daily](https://www.daily.co/) is a platform for seamless video and audio chat. The Daily plugin for Bubble, maintained by [Lunch Pail Labs](https://www.lunchpaillabs.com), makes it easy to integrate real-time communication into your app.

## What's in the Plugin?

![demo](https://res.cloudinary.com/dpfmaccxx/image/upload/v1727828385/chatplugindemo_yxpv8u.gif)

The Daily plugin provides all the essential tools to bring high-quality video and audio calls into your Bubble app. Whether you need simple conferencing or advanced features, this plugin has you covered.

### Core Features (Free Version)

The [**free version**](https://bubble.io/plugin/daily-video-chat-1622494385281x917140771709124600) includes:

- **Video and Audio Calls**: Host crystal-clear video and audio calls effortlessly.
- **Room-Based Conferencing**: Collaborate with multiple participants in dynamic rooms.
- **Participant Control**: Manage calls with mute, screen sharing, and participant management.
- **Live Streaming & Recording**: Stream meetings live or save them for later review.

### Advanced Features (PLUS Version)

The [**PLUS version**](https://bubble.io/plugin/daily-video--audio-plus-1727441795463x240182085842370560) unlocks additional tools for enhanced calls:

- **Transcription**: Automatically generate real-time captions to improve accessibility and capture key moments.
- **Custom Integrations**: Embed powerful apps like Miro for brainstorming, Google Docs for collaboration, or Spotify for shared playlists directly within your calls.

## Installation and Setup

1. Install the [**free**](https://bubble.io/plugin/daily-video-chat-1622494385281x917140771709124600) or [**PLUS**](https://bubble.io/plugin/daily-video--audio-plus-1727441795463x240182085842370560) version based on your needs.
2. **Create a Daily Account**: [Sign up for Daily](https://dashboard.daily.co/u/signup).
3. **Get Your API Key**: Head to the [Daily Developer Dashboard](https://dashboard.daily.co/developers) to grab your API key.
4. **Configure the Plugin**: Enter your API key in the Bubble plugin settings, starting with `Bearer` (e.g., `Bearer 4ffad...`).

Once installed, you’re ready to start hosting calls, managing participants, and leveraging advanced features.
