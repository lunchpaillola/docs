### Overview

The Daily plugin for Bubble makes it easy to add robust video and audio chat functionality to your app. With just a few steps, you can start offering seamless video communication within your platform.

### Quick Setup Guide

1. **Create a Daily Account:**

   - If you don’t have one yet, [sign up for a Daily account](https://dashboard.daily.co/u/signup).

2. **Retrieve Your API Key:**

   - Log in to your [Daily developer dashboard](https://dashboard.daily.co/developers) and copy your API key.

3. **Configure the Plugin:**

   - In Bubble, navigate to the plugin settings.
   - Enter your Daily API key into the API key section, prefixing it with `Bearer` (e.g., `Bearer 4ffad...`).

4. **Start Building:**
   - You’re now ready to use the Daily plugin in your workflows! Begin creating rooms, managing participants, and more using the plugin’s built-in actions.
