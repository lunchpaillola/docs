## What is Daily?

[Daily](https://www.daily.co/) is a platform for seamless video and audio chat. The Daily plugin for Bubble, maintained by [Lunch Pail Labs](https://www.lunchpaillabs.com), makes it easy to integrate real-time communication into your app.

## What's in the free plugin?

The [**free plugin**](https://bubble.io/plugin/daily-video-chat-1622494385281x917140771709124600) covers core video and audio use cases, including:

- **Video and audio calls**: Create and manage high-quality video calls.
- **Conferencing**: Manage participant interactions within rooms.
- **Participant control**: Mute, screen share, and more.
- **Live streaming and recording**: Stream meetings live or record them for later.

## What's in the plus plugin?

The [**plus plugin**](mailto:help@lunchpaillabs.com) includes everything in the free version, plus additional features for more advanced use-cases:

- [**Transcription**](/bubble/daily/how-tos/transcribe-a-meeting): Get real-time captions for your calls.
- [**Custom integrations**](/bubble/daily/how-tos/add-custom-integrations): Embed tools like Miro, Spotify and Google Docs into your meetings.

## Installation and Setup

You can install either the **free** or **plus** version of the Daily plugin:

- [Install the Free plugin](https://bubble.io/plugin/daily-video-chat-1622494385281x917140771709124600)
- [Install the Plus plugin](mailto:help@lunchpaillabs.com)

1. **Create a Daily Account**: [Sign up for Daily](https://dashboard.daily.co/u/signup).
2. **Get Your API Key**: Go to the [Daily Developer Dashboard](https://dashboard.daily.co/developers) and copy your API key.
3. **Configure the Plugin**: In your Bubble app, paste the API key into the plugin settings, prefixed with `Bearer` (e.g., `Bearer 4ffad...`).

Once set up, you can start creating rooms and managing participants right away.
