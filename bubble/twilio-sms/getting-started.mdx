This guide will help you get started with installation, setup, and your first SMS integration.

## What's the plugin?

The Twilio SMS plugin by Lunch Pail Labs allows you to send SMS messages from your Bubble app using Twilio's powerful API. This guide will walk you through the installation, setup, and sending your first SMS message.

![demo](https://res.cloudinary.com/dpfmaccxx/image/upload/v1728496716/twilio_sms_d80ydu.gif)

## Installation

### 1. Prerequisites

Ensure you have a [Twilio account](https://www.twilio.com/try-twilio) with SMS capabilities.  
Ensure you have a [Bubble account](https://bubble.io/login?mode=signup).

### 2. Install the Plugin

Navigate to the [Bubble Plugin Marketplace](https://bubble.io/plugin/twilio-sms-by-lunch-pail-labs) and install the Twilio SMS plugin by Lunch Pail Labs into your desired Bubble app.

### 3. Verify Installation

Confirm that the plugin appears in your list of installed plugins.

### 4. Input Plugin Keys

Go to the Twilio SMS plugin settings from your Bubble dashboard.  
Add your Twilio Auth Token and Twilio Account SID, which you can find in your [Twilio Console](https://www.twilio.com/console).

## Using the Plugin

This plugin offers a single action to send an SMS message. Learn more about this action in the [plugin reference](/bubble/twilio-sms/send-sms)

## Next Steps

Now that you're set up, you're ready to start sending SMS messages from your Bubble app using the Twilio SMS plugin!
