## Parameters and Customization

This component is for the audio call and represents a view of a type of participant. You can configure the following fields:

| **Setting**            | **Description**                                                                                                                                                                                                                                              |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **App ID**             | Your TalkJS app ID is found on the Settings page of your [TalkJS Dashboard](https://talkjs.com/dashboard).                                                                                                                                                   |
| **ID**                 | The user ID of the local user joining the conversation. You can use Adalo’s ID field.                                                                                                                                                                        |
| **Name**               | The user's name or username—use whatever you want others to see when communicating with this user.                                                                                                                                                           |
| **Photo**              | The photo or avatar of this user. Shown to others in conversations with this user.                                                                                                                                                                           |
| **Role**               | A user role that you can set up from the Roles page of your TalkJS dashboard. Make sure the role corresponds precisely to a role name you chose in the dashboard.                                                                                            |
| **Loader Color**       | Optional parameter to set the color of any loaders.                                                                                                                                                                                                          |
| **Start in chat view** | Starts the component in a chat view instead of the inbox view; add participants if enabled.                                                                                                                                                                  |
| **Inbox Header Color** | Optional parameter for custom themes, specify inbox header background color for Android and iOS. Web inherits color automatically.                                                                                                                           |
| **Inbox Font Color**   | Optional parameter for custom themes, specify inbox header font color for Android and iOS. Web inherits color automatically.                                                                                                                                 |
| **Participants**       | Any participants to add to the conversation should be a list of users, and for each participant, you can specify the ID, Email, Photo, or Role. If no participants are added, it will show the inbox view for all the conversations the user is involved in. |
